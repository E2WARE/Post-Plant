# Post Plant

This project is a mobile application developed using Flutter. The application fetches data from Firebase database and follows MVVM architecture.

## Features
- Firebase database integration
- MVVM architecture
- User-friendly interface

## Technologies Used
- Flutter
- Firebase
- MVVM
- JSON
- ADMOB
